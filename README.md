# Quiz de Perfil Financeiro

Projeto em Python que identifica o perfil financeiro do usuário com base em respostas a perguntas comportamentais.

## Objetivo

Este projeto foi desenvolvido para praticar:

- Lógica de programação
- Estruturas condicionais
- Entrada e saída de dados
- Organização de código em Python

## Como funciona

O usuário responde a uma série de perguntas relacionadas a hábitos financeiros.

Com base nas respostas, o sistema classifica o perfil como:

- PROJETO CAOS
- MEMBRO DO CLUBE
- LOBO DE WALL STREET
